Mark tag

What is the mark tag in HTML?

  • HTML <mark> tag is used to mark or highlight the important text relevant to the user in the webpage.
  • By default, it uses yellow color in the background
  • but using CSS properties, we can change background and text colors, the shape of the highlight, etc.
  • mark tag is in html 5

mark tag Syntax:

<mark>text....</mark>
