Address tag

  • The HTML <address> tag defines contact information for the author or owner of a webpage.
  • it can contain various contact details such as an email address, URL, physical address, mobile number, social media, etc.
  • The text inside the <address> tag will be displayed in italic format.
  • Some browsers add a line break before and after the address element
  • address tag is is html 5

HTML address Tag Usage :

  • Use <address> for showing details like organization name, website, and contact info.
  • Employ this tag to credit the writer of an article, making it clear who created it.
  • Drop <address> tag in the footer to share essential details about the site or author on every page.
