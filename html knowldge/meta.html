What is Meta tag?

  • The HTML <meta> tag provides metadata or information about the HTML document.
  • It includes information like character set, description, keywords, and other important details.
  • <meta> tags always go inside the <head> element.
  • It helps in defining the page’s title, encoding, author, and viewport settings, etc.
  • These tags are not visible on the web page but play a vital role in structuring and categorizing content for browsers and search engines.

There are various other ways to implement the <meta> tag that can use the following attribute values:

  1. Highlighting Important Keywords
  2. Providing a Description of the web page
  3. Document Revision Date
  4. Automatic Refresh
  5. Specifying Author of the Webpage
