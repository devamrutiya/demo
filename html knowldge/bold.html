BOLD Tag

What is Bold tag?

  • The <b> tag specifies bold text without any extra importance.
  • text within <b> displays in a bold style.
  • ’s a container tag with an opening, content, and closing.
  • It’s similar to <strong>, which highlights text for importance, but <b> doesn’t add significance, just makes text appear bold.
  • bold tag is in html 4

bold tag syntax:

<b> text </b>
