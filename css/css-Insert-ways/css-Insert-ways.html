Three Ways to Insert CSS

There are three ways of inserting a style sheet:

  • Inline CSS
  • Internal CSS
  • External CSS
  1. Inline CSS
  • CSS styles can be added inside the HTML tags.
  • It is one of the preliminary techniques used to write CSS in HTML. Further, with a high amount of CSS, it can lead to confusion.
  • To deal with this conspiracy further methods are implemented.

syntax for inline CSS

<p style="color:"red">dev amrutiya</p>

  1. Internal CSS
  • CSS styles are added in the HTML file by writing inside the <style> </style> tag. This is a slightly efficient method of including CSS in HTML.
  • Note: Add <style> </style> tag before the body tag as the compiler reads the code line wise it will be effective considering the runtime of the code.

  1. External CSS
  • In this type, we create an external stylesheet which means a separate file and then we link it to HTML.
  • CSS file is written separately in a .css file extension and linked to the HTML file using the <link> tag.
